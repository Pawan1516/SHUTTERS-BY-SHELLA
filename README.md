# 📸 SHUTTERS BY SHELLA

A professional photography portfolio and booking platform built with the **MERN Stack**.

## 🚀 Tech Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas
- **Media Storage**: Cloudinary CDN

## ✨ Features
- 🎬 Cinematic hero slider with real portfolio photos
- 📁 Gallery with category filtering (Wedding, Events, Portraits, Pre-wedding)
- ⭐ Client review system with admin approval
- 🔐 Secure admin dashboard
- 📤 Drag & drop media upload (Images & Videos)
- 📱 Mobile-first design
- 💬 WhatsApp & Call integration

## ⚙️ Setup

### Backend
```bash
cd backend
npm install
# Create a .env file with your keys (see .env.example)
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables
Create `backend/.env` with:
```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```
